# LibreGame
LibreGame pretende ser un motor de videojuegos basado en la libreria de Ogre 3D, y otras mas que ayuden a dar una base solida a su interfaz de desarrollo utilizando solo librerias opensource.
*Motor de renderizado > Ogre3D
*Motor de Fisica > Bullet Physics
*Audio > OpenAL
*Formato de audio > vorbis
*GUI > ImGui
*Exportar > Android
